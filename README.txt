Neil Tobin Napase - Project 2 (Angry Birds)

Features:
- Textured plane where objects are placed and move
- Cannon that shoots projectiles in direction that cannon is facing
- Phyijs physics-based 3D world
- Sounds and optional music
- 9 Independent targets that will add to score based on different collisions
- 3D text
- Score box and Help box for play keys using HTML and CSS (name in the help box)
- 3 Changeable perspectives
- Reset button, keeps high score from previous games
